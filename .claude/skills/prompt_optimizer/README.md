# Prompt Optimizer

Transform your prompts and instruction files into highly effective versions that work better with Claude.

## What It Does

Takes your prompt or guideline document and improves it using proven techniques from Claude's official documentation. You get back an optimized version plus a detailed explanation of what was changed and why.

**Just ask your AI assistant:**

- "Optimize this prompt: [paste your prompt]"
- "Improve my .cursorrules file"
- "Review this system prompt and make it better"
- "How can I make this prompt more reliable?"

**That's it!** The AI will automatically:
1. Analyze your prompt or document
2. Apply best practices for clarity, consistency, and reliability
3. Return the improved version with detailed rationale
